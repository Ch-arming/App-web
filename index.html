⚙️ Configuración API

🔑 Clave API de Gemini

Para usar el asistente IA, necesitas una clave API de Google Gemini.
Cómo obtenerla:

  1. Ve a Google AI Studio
  2. Inicia sesión con tu cuenta de Google
  3. Haz clic en "Create API Key"
  4. Selecciona un proyecto o crea uno nuevo
  5. Copia la clave generada y pégala aquí
⚠️ Nota de Seguridad: Tu clave API se guarda solo en tu dispositivo.
